# Accounting AI Panel

A modern, responsive React application featuring an aesthetic glassmorphism UI with light and dark theme support.

## Recent Features

- **Modern Glassmorphism UI:** Both the Login and Dashboard pages have been completely redesigned using modern CSS techniques, featuring blurred backgrounds, sleek gradients, and dynamic hover animations.
- **Light/Dark Theme Toggle:** Users can instantly switch between light and dark modes. The preference is saved locally (`localStorage`) and instantly applied to CSS root variables without slow transition delays.
- **Enhanced Login Page:** Features animated background blobs, gradient typography, and state-of-the-art interactive input fields.
- **Interactive Dashboard:** Includes a responsive sidebar, statistics cards, and mock charts styled to adapt perfectly depending on the active theme mode.

## Development

To start the development server, run:
```bash
npm start
```

## Structure

- `src/pages/Login.jsx` & `.css`: Authentication screen.
- `src/pages/Dashboard.jsx` & `.css`: Main application layout and sidebar.
- `src/index.css`: Global base styles and root theme variables.
